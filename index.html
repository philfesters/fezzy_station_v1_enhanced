⚡ ONE‑LINE CLONE & LAUNCH

git clone https://github.com/philfesters/fezzy_station_v1_enhanced.git

Then cd fezzy_station_v1_enhanced && bash fezzy_station_v1.sh

⚡ ADD GLOBAL ALIAS (type fezzy from anywhere)

Open your bashrc:

nano ~/.bashrc

Add this line at the bottom:

alias fezzy='bash ~/fezzy_station_v1_enhanced/fezzy_station_v1.sh'

Save (Ctrl+O, Enter, Ctrl+X), then reload:

source ~/.bashrc

Now type fezzy anywhere to launch the station.

📜 nano QUICK GUIDE

  • nano filename – open a file
  • Ctrl+O – save (press Enter to confirm)
  • Ctrl+X – exit nano
  • Ctrl+W – search inside the file
  • Ctrl+K – cut a line, Ctrl+U – paste it
